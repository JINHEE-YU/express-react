# 프로젝트 구조
## client
- React 프로젝트
- Port : 3000
## server
- Express 프로젝트
- Port : (prod)80 / (dev)5000
# 명령어
##  실행
- product환경 실행 `npm run start`
- development 환경 실행 : `npm run dev`
## 주요 패키지
* npm-run-all : npm-scripts를 여러개 동시에 실행할 수 있게 하는 모듈
* nodemon : node.js 서버 사이드에 변경이 생길 때 자동으로 재실행해서 개발을 도와주는 모듈
